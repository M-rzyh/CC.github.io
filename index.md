# Welcome to the Compute Canada Beginner Guide

This website is a practical, step-by-step onboarding guide for new graduate students using the **Digital Research Alliance of Canada (formerly Compute Canada)** HPC systems.

Whether you're doing machine learning, simulations, data processing, or GPU workloads, this page walks you through everything from account creation to running your first Slurm job.
